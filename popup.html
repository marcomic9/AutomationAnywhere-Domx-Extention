DomX Inspector

Generate robust selectors for Automation Anywhere

How to use

  • Click "Start Picking" to activate picker mode
  • Hover over elements to highlight them
  • Click any element to capture it
  • Press Escape to cancel picker mode
  • Copy generated DomX selectors for AA

AA Notes

  • AA uses 1-based indexing (not 0-based)
  • Use (//xpath)[N] for positional matching
  • XPath 1.0 only - no CSS selectors
