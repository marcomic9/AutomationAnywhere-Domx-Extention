Test Instructions

  1. Load the DomX Inspector extension in Chrome (chrome://extensions → Developer mode → Load unpacked)
  2. Click the extension icon → "Start Picking" to activate picker mode
  3. Hover over elements — they highlight with a blue border
  4. Click any element to capture it and generate selectors
  5. Press Escape to cancel picker mode
  6. Click "Validate" on any selector to test it — matched elements highlight green
  7. Test the iframe element to see iframe detection warning
  8. Try dynamic ID buttons (section 2) — hardened contains() selectors auto-generate
  9. Try sibling/relational elements (section 5) — label→input patterns appear
